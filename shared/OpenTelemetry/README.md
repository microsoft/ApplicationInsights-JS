# Microsoft Application Insights JavaScript SDK - Core

[![GitHub Workflow Status (main)](https://img.shields.io/github/actions/workflow/status/microsoft/ApplicationInsights-JS/ci.yml?branch=main)](https://github.com/microsoft/ApplicationInsights-JS/tree/main)
[![npm version](https://badge.fury.io/js/%40microsoft%2Fotel-core-js.svg)](https://badge.fury.io/js/%40microsoft%2Fotel-core-js)

OpenTelemetry Core provides the basic implementation for supporting the [CNCF OpenTelemetry specifications](https://github.com/open-telemetry/opentelemetry-specification)

## Build & Test:
- Install: `$ npm install`
- Build: `$ npm run build`
- Test: `$ npm run test`
